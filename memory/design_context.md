## 3D 도면 작업

**프로세스:**
1. gsk analyze로 전체 읽기 → 집기명/위치/치수 파악
2. 배치 위치 절대 변경 금지
3. 모든 집기 mm 단위 치수 표기
4. 생성 후 원본 비교 (종류/개수/위치/치수/누락)
5. gsk upload로 다운로드 링크 + 비교표 제출

**반복 금지 오류:**
- 싱크대 누락
- 냉장고 별도 개체 누락
- 냉동고 치수 혼용 (650mm ↔ 850mm)
- 분전함 위치 이탈
- 기둥 좌하단 누락
- 3단선반 위치 오류 (소분대 오른쪽)
